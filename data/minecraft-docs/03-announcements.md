# 公告：文件系統上線

Minecraft 教學與公告頁面已正式建立。

## 新增內容

- GitHub 風格文件版面
- 自動讀取資料夾內所有 Markdown 文件
- 左側文章列表
- 文章搜尋
- 右側章節目錄
- 手機版文章選單

## 如何新增公告

在以下資料夾建立新的 Markdown 文件：

```text
data/minecraft-docs/
```

建議使用編號控制排序：

```text
01-start.md
02-rules.md
03-announcements.md
04-new-feature.md
```

<div class="notice">
檔案上傳到 GitHub 後，重新整理頁面便會自動出現在左側列表。
</div>
