# 新手開始指南

歡迎來到葉離 Minecraft 教學中心。

這裡會整理伺服器的玩法、指令、規則和最新公告。你只需要在 `data/minecraft-docs` 資料夾加入新的 `.md` 文件，網頁就會自動讀取並顯示。

## 如何加入伺服器

1. 開啟 Minecraft。
2. 選擇多人遊戲。
3. 新增伺服器。
4. 輸入伺服器位址。
5. 點擊加入。

<div class="info">
<strong>提示：</strong>伺服器位址可以在正式開放後放在這裡。
</div>

## 常用指令

| 指令 | 功能 |
|---|---|
| `/spawn` | 返回出生點 |
| `/home` | 返回家 |
| `/sethome` | 設定家 |
| `/tpa 玩家名稱` | 傳送請求 |

## 注意事項

請先閱讀伺服器規則，再開始遊玩。
